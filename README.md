A Python-based terminal crypto converter with Android (Pydroid3) and Windows support. It fetches real-time prices and historical trends (24h, 7d, 1m, 3m, 1y, all-time) from CoinGecko and CoinCap (with automatic failover), supports user-selected fiat currency, and displays network/exchange fees (with fiat equivalents). Fully navigable via keyboard, paginated top-100 list, search by symbol/ID/name, and customizable API prioritization.
